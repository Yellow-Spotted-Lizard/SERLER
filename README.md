# SERLER
Software Engineering Research Lab Evidence Repository (SERLER)

# Quick Start

Back-end project: `api`
Front-end project: `client`

Run full-stack:

    npm start

Run back-end only:

    cd api
    npm install
    npm start

Replace `npm start` with `DEBUG=api:* npm start` to see more logs

Run front-end only:

    cd client
    npm install
    npm start
